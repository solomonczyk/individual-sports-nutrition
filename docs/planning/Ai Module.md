# AI-модуль приложения

## 1. Сбор и обработка данных пользователя

* Вводимые пользователем данные: пол, возраст, вес, рост, уровень активности, цели, заболевания, противопоказания, лекарства
* Нормализация и валидация данных
* Шифрование и безопасное хранение персональных данных

## 2. Логика подбора питания

* Расчёт потребностей организма в макро- и микроэлементах
* Формирование индивидуального плана питания и добавок
* Определение оптимальных дозировок и частоты приёма

## 3. Проверка противопоказаний

* Сопоставление выбранных добавок и продуктов с заболеваниями и лекарственными средствами
* Предупреждения и блокировки небезопасных комбинаций

## 4. Персонализация рекомендаций

* Подстройка планов под пол, возраст, цели и уровень физической нагрузки
* Учёт изменений в параметрах и корректировка плана в реальном времени

## 5. Интеграция с локальными брендами и магазинами

* Сопоставление доступных продуктов с рекомендациями
* Указание местоположения и цены продуктов
* Возможность прямого заказа через партнёрские API

## 6. Используемые технологии

* ML/AI: модели для расчёта персональных рекомендаций
* LLM: генерация объяснений и советов для пользователя
* Сервисы: Python (TensorFlow/PyTorch), Node.js для API, PostgreSQL/MongoDB для хранения данных

## 7. Безопасность и этика

* Минимизация рисков для здоровья пользователей
* Чёткая маркировка советов как рекомендаций, а не медицинских предписаний
* Соблюдение законодательства Сербии по защите данных
