# UI/UX прототипы мобильного приложения

## 1. Принципы дизайна

* Минимализм и удобство использования
* Лёгкая навигация между разделами
* Чёткая визуальная иерархия информации
* Использование цветовой схемы для обозначения статуса и рекомендаций

## 2. Основные экраны

### 2.1 Онбординг

* Введение в функционал приложения
* Краткое объяснение ценности персонализированного подхода
* Настройка базовых уведомлений и параметров

### 2.2 Ввод параметров здоровья

* Формы для ввода данных: пол, возраст, рост, вес, уровень активности, цели, заболевания, лекарства
* Подсказки и валидация данных в реальном времени

### 2.3 Рекомендации и персонализированный план

* Интерактивная карточка плана питания и добавок
* Возможность детального просмотра дозировок и состава
* Кнопка «купить» для интеграции с партнёрскими магазинами

### 2.4 Трекинг прогресса

* Графики и визуализация изменений веса, физической формы, параметров здоровья
* История прогресса и предыдущих планов

### 2.5 Корректировки и уведомления

* Система уведомлений о необходимых изменениях в плане
* Предложения по улучшению питания и активности

### 2.6 Партнёрские товары и бренды

* Каталог доступных добавок с ценой и местоположением магазинов
* Интерактивные фильтры по бренду, типу продукта, наличию

## 3. Прототипирование

* Использование Figma / Adobe XD для создания интерактивных макетов
* Тестирование сценариев пользователей с обратной связью
* Итеративная доработка на основе тестов UX
