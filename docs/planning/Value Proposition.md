# Value Proposition (УТП)

## Главная ценность

Персонализированное спортивное питание с учётом здоровья, уровня нагрузки, целей, противопоказаний и локальной специфики сербского рынка.

## 1. Максимальная персонализация

Приложение формирует рекомендации на основе:

* пола
* возраста
* уровня физической активности
* текущих целей (масса, сушка, выносливость)
* заболеваний
* противопоказаний
* лекарственных взаимодействий

## 2. Безопасность и медико-ориентированный подход

Система исключает добавки, которые:

* могут конфликтовать с лекарствами
* запрещены при определённых заболеваниях
* имеют риски побочных эффектов

## 3. Адаптация под сербский рынок

* учёт доступных брендов и локальных магазинов
* интеграция с локальными e-commerce площадками
* соответствие сербским стандартам качества и составов

## 4. Полный цикл сопровождения пользователя

* подбор
* трекинг прогресса
* корректировки рекомендаций
* адаптация под изменения состояния организма

## 5. Экономия времени и средств

Пользователь получает только те добавки, которые действительно нужны, без лишних затрат.

## 6. Удобство и интуитивность

Простой интерфейс, быстрые рекомендации и понятная логика работы для новичков и опытных спортсменов.
