# Individual Sports Nutrition Mobile App

Мобильное приложение для персонализированного подбора спортивного питания.

## Технологии

- React Native (Expo)
- TypeScript
- Expo Router (file-based routing)
- React Query (TanStack Query)
- NativeWind (Tailwind CSS)
- Zustand (state management)
- React Hook Form + Zod

## Установка

```bash
npm install
```

## Запуск

### Development
```bash
npm start
```

### iOS
```bash
npm run ios
```

### Android
```bash
npm run android
```

### Web
```bash
npm run web
```

## Структура проекта

```
app/                    # Expo Router (file-based routing)
src/
├── components/         # Переиспользуемые компоненты
├── screens/            # Экраны приложения
├── services/           # API клиенты
├── store/              # Zustand stores
├── types/              # TypeScript типы
├── utils/              # Утилиты
└── hooks/              # Custom hooks
```
